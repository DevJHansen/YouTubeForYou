# YouTube ForYou

**Take back your attention.** A Chrome extension that blocks YouTube's distracting homepage and replaces it with a simple search bar.

---

## The Problem

YouTube's homepage is engineered to keep you scrolling. You go to search for one thing, and 30 minutes later you're watching videos you never intended to watch.

## The Solution

ForYou intercepts the homepage and shows you a clean, focused search interface instead. No recommendations. No rabbit holes. Just a simple question:

> *"What did you come here to watch?"*

The irony? By removing YouTube's "For You" recommendations, it actually works **for you**.

---

## Features

- **Homepage Blocker** — Replaces the feed with a distraction-free overlay
- **Quick Search** — Jump straight to what you're looking for
- **Escape Hatch** — "Show homepage anyway" button when you actually want to browse
- **Seamless** — Works with YouTube's single-page navigation

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked**
5. Select the extension folder

---

## Usage

Navigate to `youtube.com`. Instead of the recommendation feed, you'll see a search bar.

Type what you're looking for and press enter. That's it.

Want to browse the homepage? Click **"Show homepage anyway"** — it'll stay visible for your current session.
