# ForYou

A Chrome extension that blocks YouTube's distracting homepage and replaces it with a simple search bar.

## Why?

YouTube's homepage is designed to keep you browsing. When you go to YouTube to search for something specific, you often get sidetracked by recommended videos. ForYou removes that temptation by showing you a focused search interface instead.

The irony? By removing YouTube's "For You" recommendations, it actually works *for you*.

## Features

- Blocks the YouTube homepage with a clean, dark-themed overlay
- Simple search bar to find what you came for
- "Show homepage anyway" button if you want to browse
- Works seamlessly with YouTube's navigation

## Installation

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked**
5. Select the `ForYou` folder

## Usage

Just go to youtube.com. Instead of the recommendation feed, you'll see a search bar. Type what you're looking for and hit enter.

If you actually want to browse the homepage, click "Show homepage anyway" - this will show the normal page for your current session.
